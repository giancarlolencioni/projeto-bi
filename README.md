# projeto-bi

## Descrição
Este projeto tem como objetivo realizar análises de dados utilizando Power BI a partir de um dataset de vendas.

## Análise realizada
Na primeira etapa do projeto foi criada uma medida em DAX para calcular o preço médio dos produtos. Essa medida foi utilizada para gerar uma visualização que mostra o preço médio por linha de produto.

## Ferramentas utilizadas
- Power BI
- DAX
- Dataset em formato CSV

## Estrutura do repositório
- dataset/ -> arquivo CSV com os dados de vendas
- powerbi/ -> arquivo do relatório do Power BI
- dax/ -> arquivo com a medida DAX utilizada
